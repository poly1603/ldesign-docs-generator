### 配置建议

**小型项目** (<100页):
```typescript
{
  plugins: [markdownPlugin()],
  markdown: { lineNumbers: true },
}
```

**中型项目** (100-500页):
```typescript
{
  plugins: [markdownPlugin(), typedocPlugin()],
  cacheDir: '.cache/docs',
  build: { codeSplit: { enabled: true } },
}
```

**大型项目** (>500页):
```typescript
{
  plugins: [/* 所有插件 */],
  cacheDir: '.cache/docs',
  build: {
    codeSplit: { enabled: true },
    imageOptimization: { enabled: true },
    prefetch: { enabled: true },
  },
  pwa: { enabled: true },
}
```
